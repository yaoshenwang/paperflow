# Demo Project

This is a file-first Paperflow project example.

The project is meant to mirror the new `@paperflow/exam-markdown` scaffold:

- `paper.qmd` as the paper entry file
- `questions/*.md` as individual question sources
- `_quarto.yml` as the Quarto project config
- `templates/` for the template extension
- `.paperflow/` for local cache and preview artifacts
