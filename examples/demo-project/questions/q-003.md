::: {.stem}
已知函数 $f(x)=x^2-2x+3$。

（1）求函数 $f(x)$ 的最小值；

（2）判断 $f(x)$ 在 $[1,+\infty)$ 上的单调性，并证明。
:::

::: {.answer}
（1）$f(x)=(x-1)^2+2$，最小值为 $2$。
（2）$f(x)$ 在 $[1,+\infty)$ 上单调递增。
:::

::: {.analysis}
配方法可得 $f(x)=(x-1)^2+2$，顶点为 $(1,2)$。

对于单调性证明，取 $x_1>x_2\ge 1$，则 $f(x_1)-f(x_2)>0$。
:::
