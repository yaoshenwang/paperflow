::: {.stem}
已知函数 $f(x)=x^2-2x+1$，则下列结论正确的是：
:::

::: {.options}
- A. 最小值为 0
- B. 最大值为 0
- C. 图像开口向下
- D. 对称轴为 $x=1$
:::

::: {.answer}
A、D
:::

::: {.analysis}
$f(x)=(x-1)^2$，开口向上，最小值为 0，对称轴为 $x=1$。
:::
